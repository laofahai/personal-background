# Personal Background

This repository is a personal background kit and data home. It has not been bootstrapped yet.

To set it up, open this repo in your AI agent and paste the contents of [`bootstrap/PROMPT.md`](bootstrap/PROMPT.md). The agent will generate a personalized `AGENTS.md`, scaffold your data files, install skills, and register the MCP server.

The canonical agent instructions are at [`kit/templates/AGENTS.md`](kit/templates/AGENTS.md).
