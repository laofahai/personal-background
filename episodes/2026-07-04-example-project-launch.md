# Launching the personal-background open-source project

## Context

Wanted to build a unified personal background management system that AI agents could reference in future research, decisions, and conversations.

## Event

Started designing and building the `personal-background` repository using a local-first, markdown-based approach.

## Decision

- Unified core: profile + preferences + constraints + episodes + notes
- Prioritize support for Factory, Claude, and Codex agents
- AI-driven skills as primary, scripts as helpers
- Use a private GitHub repository as the default sync target

## Outcome

Project skeleton completed, including three Factory Droid skills and cross-agent adapter files.

## Reflection

Local-first and privacy-friendly are key selling points. The AI-as-librarian approach is more sustainable than manual maintenance.
