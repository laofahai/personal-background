# Personal Background - Bootstrap Prompt

Paste this whole block into your AI coding agent from inside the cloned repo.

---

You are setting up my personal-background system. This repository is BOTH a kit and my
git-synced markdown data home. Do the following, asking me for confirmation before any write:

1. **Ask me for my preferred language** (e.g. en, zh). This is the language all my user data
   will be written in. Remember it for the rest of this setup.
2. Read `kit/manifest.yml`, `methodology.md`, and `kit/templates/AGENTS.md` to understand the system.
3. Create `.pbg/settings.yml` by copying `kit/config/settings.example.yml` and filling in:
   - `preferred_language`: my answer from step 1
   - `repo_path`: the absolute path of this repo (detect it)
   - `kit_version`: the `kit_version` from `kit/manifest.yml`
4. Generate the root `AGENTS.md` by copying `kit/templates/AGENTS.md` and replacing
   `{{REPO_PATH}}` with the absolute repo path. If a root `AGENTS.md` already exists and is
   not the placeholder, show me a diff and ask before overwriting. This file is user-owned;
   future kit upgrades will not touch it.
5. If they do not already exist, create my core files from `kit/templates/`:
   `profile.md`, `preferences.md`, `constraints.md`, and the directories
   `episodes/`, `notes/`, `raw/public/`, `raw/private/`, `archive/`.
   NEVER overwrite any of these if they already exist.
6. Detect which agent you are (Factory/Claude/Codex/Cursor/...). Install each skill in
   `kit/skills/*/SKILL.md` into this agent's skill directory. Where a skill references
   the repo path, it reads `.pbg/settings.yml` `repo_path`, so no path hardcoding is needed.
7. Register the MCP server so I can read/write my background from ANY project. Choose the runtime I have installed:
   - **Bun (no build step):** `cd mcp && bun install`, then
     `droid mcp add personal-background --env PERSONAL_BACKGROUND_DIR=<repo_path> -- bun <repo_path>/mcp/src/server.ts`
   - **Node:** `cd mcp && npm install && npm run build`, then
     `droid mcp add personal-background --env PERSONAL_BACKGROUND_DIR=<repo_path> -- node <repo_path>/mcp/dist/server.js`
   - Adapt the command to my actual agent if different.
8. Tell me I can now either run the `complete-profile` skill for guided onboarding, or
   just talk to you and say "remember this" to capture episodes/notes.

Rules:
- Framework-owned files (`kit/`, `methodology.md`, `bootstrap/`, `README.md`, `README.zh.md`,
  `docs/`, `LICENSE`) are yours to install/read; user-owned files are never overwritten.
- `AGENTS.md` (root) is user-owned once generated by this bootstrap; future kit upgrades will not overwrite it.
- Write my data in my preferred language. Never read or upload `raw/private/`.
- Show me a short plan before writing anything.
